# Portfolio Website - Abdullah Maulana Putra

This is the personal portfolio website of **Abdullah Maulana Putra**, a Data Analyst and Web Developer.

## Features
- **Personal Profile**: Bio, Education, and Organization history.
- **Project Showcase**: Display of Data Analysis, Web Development, and AI projects.
- **Skills & Tech Stack**: Overview of technical expertise.
- **Certificate Gallery**: Verification of achievements and courses.
- **Contact Form**: Direct messaging capability.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS
- **Animations**: AOS (Animate On Scroll)
- **Icons**: Lucide React
- **Data**: Local JSON (formerly Supabase)

## Setup & Running
1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Run development server:
    ```bash
    npm run dev
    ```

## Customization
To update project data, edit `src/Pages/Portofolio.jsx`.
To update personal details, edit `src/Pages/Home.jsx` and `src/Pages/About.jsx`.
